Quick Start

  1. Open the app and click ⚙ in the top-right corner.
  2. Enter your API key under the Keys tab.
  3. Start a conversation — the app is ready to use immediately.

Minimum Requirements

  • macOS 10.15 or later
  • An active API key from a supported AI provider

FAQ

Where do I get an API key?

You can obtain an API key from your preferred AI service provider. The app supports multiple providers — configure the key under Settings → Keys.

Where is my data stored?

All conversations and files are stored locally on your device. Nothing is uploaded to our servers. Your API key is saved in ~/.deepdesk.env on your machine.

How do I upgrade to Pro?

Visit shieldyh.com to purchase a Pro license, then activate it in the app under Settings → License.

The app says "Server Not Running"

This means the local backend process failed to start. Try:

  1. Restart the app.
  2. Make sure port 3456 is not in use by another application.
  3. If the issue persists, contact us below.

How do I use Agent mode?

Toggle the 🤖 Agent switch below the input box. The agent can autonomously use tools, run shell commands, and browse the web. First warmup takes ~25 seconds.
